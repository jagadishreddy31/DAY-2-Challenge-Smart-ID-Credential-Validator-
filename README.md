# DAY-2-Challenge-Smart-ID-Credential-Validator-
The DAY-2 challenge  is to design a basic validation system for student registration .the program takes multiple user inputs and checks whether they follow the given university rules .if all the entered details are valid , the system approves the registration ; otherwise , it rejects it.the challenge mainly focuses on understanding string operations and conditional statements without using loops , regular expressions , or advanced functions .
